### How do I get set up? ###

* Install Node.JS
* Clone this Repo
* Create a file called `token.txt` and paste your bot token in there
* Run `npm install` (NOT WORKING UNLESS YOU ARE ON DEV BRANCH)
* Run `node tombot.js`
